# PHP Playground

Start playground (Requires Docker)

- `docker compose up -d`

Stop playground (Requires Docker)

- `docker compose down`

Versions and Links

- `v8.0` [http://localhost:8000](http://localhost:8000)
- `v8.1` [http://localhost:8001](http://localhost:8001)
